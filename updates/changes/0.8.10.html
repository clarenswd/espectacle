Spectacle 0.8.10

This version of Spectacle provides the following improvements:

  • Adds consistent behavior when moving a window from 1/2 to 1/3 and finally 2/3 of a display
  • Improves support for moving windows between thirds of the display
  • Fixes quantized window (e.g. Terminal, iTerm 2, etc.) movement

Important Note: Spectacle now requires OS X 10.9 or better. Please review the README for additional information.
