Spectacle 0.8

This version of Spectacle provides the following improvements:

  • No longer resizes or moves windows that already fill the dimensions of the current display
  • Fixes an issue where moving a window backwards between thirds of the display did not always work
  • Supports increasing and/or decreasing the size of arbitrary windows

Important Note: Spectacle now requires OS X 10.7 or better. Those still using OS X 10.6 should skip this update and continue to use the 0.7 release of Spectacle.
