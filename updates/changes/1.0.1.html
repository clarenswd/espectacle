Spectacle 1.0.1

This version of Spectacle provides the following improvements:

  • Supports shortcuts saved with older versions of Spectacle
  • Fixes an issue where Spectacle would not work with certain windows

Thanks to everyone who has helped make Spectacle better by reporting issues and opening pull requests! Spectacle has been long overdue for 1.0 release and it would not have been possible without the help from so many great folks.

Important Note: Spectacle now requires OS X 10.9 or better. Please review the README for additional information.
