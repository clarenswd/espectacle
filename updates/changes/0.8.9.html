Spectacle 0.8.9

This version of Spectacle provides the following improvements:

  • Improves the quality of the status menu icon
  • Adds French and Italian translations
  • Adds preliminary support for function keys as shortcuts
  • Removes the 1px gap that can sometimes appear on the top edge of the display
  • Supports resizing windows to 1/2, 2/3, and 1/3 the height of the display

Important Note: Spectacle now requires OS X 10.9 or better. Please review the README for additional information.
