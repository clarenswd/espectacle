Spectacle 0.8.6

This version of Spectacle provides the following improvements:

  • Supports moving windows to external displays in a consistent arrangement
  • Disables Spectacle when the preferences window is in focus
  • Updates the status bar icon to look correct in Yosemite
  • Improves support for resizing quantized windows (e.g. Terminal, iTerm, etc.)

Important Note: Spectacle now requires OS X 10.7 or better. Those still using OS X 10.6 should skip this update and continue to use the 0.7 release of Spectacle.
